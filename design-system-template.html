Regras de Design

R1 — Lei dos 70/30: A Fotografia Comanda

Texto + gradientes + elementos graficos combinados nunca devem ocupar mais de 30% da altura total. Os 70% restantes devem ser fotografia limpa. O visualizador para de rolar por causa da foto, nao do texto.

R2 — Sistema de Overlay: Contraste Calculado

Overlays usam gradiente linear com opacidade descendente (92% → 75% → 0%). Altura maxima: 30% a 50% da arte. O overlay nao deve "sufocar" a foto — apenas criar contraste para o texto.

R3 — Margens Seguras (Safe Zones)

Feed (4:5): padding 5% em todos os lados. Nunca colocar texto ou logo nos 5% externos — area cortada por alguns dispositivos.

R4 — Hierarquia Tipografica

PRE-TITLE → font-body, small, ALL CAPS, wide spacing
TITLE → font-heading, large, line-height: 0.85
DESCRIPTION → font-body, small, line-height: 1.3
CTA / TAG → font-body, ALL CAPS, minimal

R5 — Logo: Presente mas Discreto

Logo no topo de cada slide, centralizado ou alinhado a esquerda. Maximo 20% da largura. Deve estar presente mas nunca ser protagonista.

R6 — Ritmo do Carrossel

Alternar posicao do overlay entre slides (bottom → top → left → right → CTA). Manter tipografia e cores consistentes. Cada slide motiva o proximo swipe.

Layout Presets

Cada preset define a sequencia de variacao de overlay para os slides. O usuario escolhe o preset ao criar o carrossel.

PresetSlide 1Slide 2Slide 3Slide 4Slide 5Quando usar
standard photo + overlay-bottom photo + overlay-top photo + overlay-left photo + overlay-right gradient-brand Conteudo geral, promocoes
editorial photo + overlay-bottom photo + overlay-left photo + overlay-bottom photo + overlay-right gradient-brand Conteudo mais sofisticado
bold gradient-brand photo + overlay-top photo + overlay-bottom photo + overlay-top gradient-brand Alto impacto, promocoes fortes

Presets sao configurados no design-system.json do projeto. Podem ser customizados por projeto.

Fluxo de Producao — Checklist

  • Carregar projeto: Verificar brand.json e design-system.json do cache local.
  • Definir tema e copys: Escrever headline + body para cada slide, respeitando tom de voz do KB.
  • Escolher preset: Selecionar sequencia de overlay (standard, editorial, bold).
  • Curadoria de imagens: Selecionar foto de fundo para cada slide via pagina de curadoria.
  • Aplicar tokens: Usar exclusivamente cores e fontes do design system.
  • Verificar R1 (70/30): Foto e o protagonista, texto e apoio.
  • Aplicar overlay (R2): Testar contraste do texto sobre o fundo.
  • Inserir logo (R5): Discreto, topo de cada slide.
  • Revisar ritmo (R6): Alternar posicao de overlay entre slides.
  • Exportar: Playwright → PNGs 1080x1350px.
